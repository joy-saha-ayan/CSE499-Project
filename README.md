# TruthLens AR

## Bangla Multimodal AI Fake News Detection

### Abstract

TruthLens AR is a research project that aims to develop a Bangla multimodal fake news detection system using artificial intelligence. The dataset consists of textual claims, images, videos, evidence documents, labels (Real/Fake), and source links collected from reliable fact-checking websites and news sources. The collected dataset will be used to train and evaluate multimodal machine learning models for fake news detection.
